# SubmitHandler

A hand-crafted Black Friday deal submission service to help indie developers and SaaS founders get maximum exposure for their Black Friday deals.

## About This Project

SubmitHandler was created to help [Mike](https://x.com/MJEIT_) kickstart his indie developer journey. This service takes the tedious work of manually submitting Black Friday deals to 23+ curated directories and handles it professionally, giving founders back hours of their time while ensuring maximum visibility during the crucial Black Friday sales period.

## What It Does

- **Hand-crafted Submissions**: Mike personally submits your Black Friday deal to 23 premium directories
- **24-Hour Turnaround**: Fast, reliable submission within 24 hours of payment
- **Detailed Report**: Comprehensive report with all submission links and timestamps
- **Curated Directories**: List carefully curated by [Ayush](https://x.com/ayushtweetshere)
- **Optional Paid Add-Ons**: 3 additional premium directories available

## Features

- Clean, modern, responsive design
- Full SEO optimization (meta tags, Open Graph, Schema.org)
- Dedicated directories page showcasing all submission targets
- FAQ section for common questions
- Contact form for inquiries
- No external dependencies - pure HTML/CSS/JS

## Technology Stack

- **Frontend**: Pure HTML5, CSS3, JavaScript (ES6+)
- **Fonts**: System font stack for optimal performance
- **SEO**: Complete meta tags, sitemap.xml, robots.txt, Schema.org structured data
- **Design**: Mobile-first responsive design with elegant animations

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/julianengel/SubmitHandler.git
cd SubmitHandler
```

2. Run a local server:
```bash
python3 -m http.server 8000
```

3. Open your browser to `http://localhost:8000`
